Security

Security at 36Blocks

We take security seriously. Every layer of 36Blocks is designed with your users' data and trust in mind.

Last updated: January 2026

Infrastructure & Hosting

36Blocks runs on enterprise-grade cloud infrastructure with automatic failover, redundancy, and geographic distribution. All services are containerised and isolated to prevent cross-tenant data leakage.

Data Encryption

All data is encrypted both in transit and at rest. We use TLS 1.2+ for all network communication and AES-256 for data stored on disk. Passwords are hashed using bcrypt and are never stored in plaintext.

Access Control & Audit Logs

Role-Based Access Control (RBAC) is built into the core of 36Blocks. Every action is logged with a full audit trail so you can see who did what, when. Logs are retained per your plan and are tamper-evident.

Responsible Disclosure

If you discover a security vulnerability, please contact us at security@36blocks.com. We will respond within 48 hours and work with you to resolve the issue responsibly.
